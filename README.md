# Mobile Banking App

Modern financial mobile application with elegant dark theme design. Features card management, transaction tracking, expenses visualization, and notifications system.

## Tech Stack

- React Native + Expo
- TypeScript
- expo-router for navigation
- expo-linear-gradient for card designs
- react-native-svg for custom icons

## Quick Start

```bash
npm install
npx expo start
```

Scan QR code with Expo Go app or run on iOS/Android emulator.
